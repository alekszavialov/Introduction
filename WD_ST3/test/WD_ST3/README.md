# WD_ST3

Draggable Speech Balloons

## Installation

### Requirements

* OS : Windows
* Web Server (XAMPP, MAMP, DENWER)
* PHP 7 and up
* GIT

`$ git clone https://github.com/alekszavialov/Introduction.git`

## Usage

```
Extract files to webserver directory.
Run your webserver.
Go to (http://{yourhost}/{yourpath}/WD_ST3/public/index.php)
```
